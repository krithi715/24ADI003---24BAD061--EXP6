This repository contains machine learning experiments focused on ensemble learning techniques and handling imbalanced data. It includes five scenarios: Bagging for diabetes prediction, Boosting (AdaBoost and Gradient Boosting) for customer churn prediction, Random Forest for income classification, Stacking for heart disease prediction, and SMOTE for fraud detection. Each scenario involves training models, comparing performance, and visualizing results using graphs such as accuracy charts, ROC curves, feature importance, and Precision-Recall curves. The project demonstrates how ensemble methods improve model performance and highlights the importance of handling class imbalance in real-world datasets.
